# ThreeJs / Webpack Template

A quick starter repo for THREE.js experiments. (WIP)

## Includes

-   Package manager animation
-   Global store for event handling
-   Shader support
-   GUI for refining

## Packages

-   Webpack
-   Three.js
-   Sass
-   Glslify
-   GSAP

## Quick Start

```bash
npm run dev
```

## Debug

Include '#debug' in the url to initiate the GUI
